4) НАСТРОИТЬ ПОДКЛЮЧЕНИЕ К MYSQL
--------------------------------
1. В корне проекта есть файл:
   .env.example

2. Сделай копию этого файла с именем:
   .env

3. Открой .env и заполни значения:

PORT=3000
MYSQL_HOST=127.0.0.1
MYSQL_PORT=3306
MYSQL_USER=root
MYSQL_PASSWORD=ТУТ_ТВОЙ_ПАРОЛЬ_ИЛИ_ПУСТО
MYSQL_DATABASE=church_prichod

Важно:
- Если у root нет пароля, оставь MYSQL_PASSWORD пустым:
  MYSQL_PASSWORD=

8) ЕСЛИ НЕ РАБОТАЕТ (КРАТКО)
----------------------------
Проблема: "не удалось подключиться к MySQL"
Решение:
- проверь, что MySQL запущен
- проверь логин/пароль в .env
- проверь порт (обычно 3306)

Проблема: Access denied for user 'root'@'localhost' (using password: YES)
Это значит: MySQL отклонил вход — в .env указан не тот пароль (или пользователь),
  что ты используешь в phpMyAdmin / при установке MySQL.
Решение: открой .env, исправь MYSQL_PASSWORD и MYSQL_USER; если пароля нет у root,
  должна быть строка MYSQL_PASSWORD= (пусто, без пробелов).

Проблема: "порт занят"
Решение:
- измени PORT в .env, например PORT=3001
- запускай снова npm start
- открывай http://localhost:3001

Проблема: "Cannot find module ..."
Решение:
- заново выполни npm install в папке проекта

Проблема: TypeError: require(...) is not a function (finalhandler / express)
Причина: сломанная папка node_modules (часто пустой файл в пакете debug).
Решение в PowerShell из папки проекта:
  Remove-Item -Recurse -Force node_modules
  npm install


9) НОВОСТИ ЧЕРЕЗ PHP (БЕЗ NODE)
--------------------------------
Файлы: public\news.php и public\includes\news_helpers.php

Как открыть:
- Скопируйте папку public на хостинг с PHP или укажите корень сайта на public в OpenServer/XAMPP.
- В браузере: http://ваш-сайт/news.php

Что нужно в php.ini:
- allow_url_fopen = On (чтобы file_get_contents мог брать страницы vob и RSS Патриархии по HTTPS).
- Расширение openssl включено (обычно уже есть).

На локальном ПК через "npm start" PHP не выполняется: откройте news.html или поднимите Apache и укажите DocumentRoot на папку public.
